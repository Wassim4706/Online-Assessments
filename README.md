Online Assessment System
A web-based online assessment platform that allows administrators to create, manage, and evaluate exams while enabling students to take assessments online in a secure and organized environment.
The system is built using HTML, CSS, JavaScript, PHP, and MySQL, providing a simple and efficient solution for managing quizzes, tests, and results.
Features

Admin Features:
Create and manage exams
Add, edit, and delete questions
Manage student accounts
View student results and performance
Automatic grading for objective questions

Student Features:
User authentication (login/register)
Take online assessments
Submit answers digitally
View results after completing exams

System Features:
Secure database storage using MySQL
Dynamic web interface using JavaScript
Server-side processing with PHP
Responsive UI using HTML & CSS

Technologies Used
Technology	Purpose
HTML	Structure of the web pages
CSS	Styling and layout
JavaScript	Client-side interaction
PHP	Backend logic
MySQL	Database management
